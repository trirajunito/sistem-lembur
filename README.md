# Sistem Lembur

Aplikasi manajemen lembur berbasis Laravel.

Fitur Utama

# Karyawan
Melakukan registrasi dan login ke sistem.
Mengajukan permohonan lembur.
Melihat riwayat pengajuan lembur.
Memantau status persetujuan lembur.
Melihat laporan lembur pribadi.

# Atasan
Melihat daftar pengajuan lembur dari karyawan.
Menyetujui atau menolak pengajuan lembur.
Memantau data lembur yang telah diproses.

# Admin / HRD
Mengelola data karyawan.
Melihat seluruh data pengajuan lembur.
Mengakses laporan lembur.
Memantau statistik lembur melalui dashboard.
